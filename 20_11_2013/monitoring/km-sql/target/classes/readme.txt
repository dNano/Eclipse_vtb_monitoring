Полная последовательность сборки скриптов такова:
1. drop.sql   -- удаление всех объектов.
2. create.sql  -- создание всех объектов.
3. fill_system_data.sql -- наполение справочников и прочих обязательных вещей.
4. fill_test_data.sql   -- наполнение тестовыми данными.

Если надо просто почистить рабочие таблицы, не пересобирая все, то достаточно выполнить скрипт fill_test_data.sql
Или вообще выполнить в нем только операторы delete.
